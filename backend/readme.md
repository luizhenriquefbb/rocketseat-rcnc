# ARCNC - Backend

This is the backend of the "RCNC" application.

## How to run

### Install dependencies

```sh
yarn install
```

### Execute

```sh
yarn execute
```
or
```sh
yarn dev
```

## Features:

- using express
- using mongoDB
- sockets to real time communication
- Upload of images
